1. Меня зовут Антон Лобков. Мой проект - сайт для создания и решения школьных тестов. Руководитель - Анатольев Алексей Владимирович, учитель информатики.

2. Объектом моего проекта является интеграция информационных технологий в образовательный процесс. Предметом же является сам сайт-конструктор. 

3. Актуальность. Во первых тесты - простой и быстрый способ проверки знаний, к тому же очень популярный. Во-вторых сейчас обучения все больше переходит в интернет.

4. Структура проекта, основными состовляющими являются фронтенд (это сайт) и бекенд (сервер). Два самых важных компонента сайта - решения тестов и авторизация. При нажатии на кнопку ответить сриазу же отправляется ответ на сервер и возвращает следующее задание, или результат если оно было последним. Авториазция работает с помощью токенов, клиент отправляет запрос на сервер, а сервер генерирует уникальный ключ - токен и возвращает клиенту, ключ хранится в браузере 30 дней, аотогм надо входить заново.

5. Технологии. HTML, CSS, JavaScript - основа всех сайтов: разметка сайта, стили и скрипты для анимаций отправки и тд. Python - язык программирования на котором написан сервер. React - JS библеотека для создания пользовательских интерфейсов. Она позволяет создавать компоненты, из которых будет состоять страница, и которыми можно удобно управлять. Flask - Python библиотека для создания сервера. SQLite, SQLAlchemy - это база данных и бибилиотека для удобного управления ею.

6. Итак начать проект я решил с создания системы аккаунтов, была создана регистрация, авторизация, а позже страница профиля. Есть 2 типа аккаунтов учителя и ученика. Там где сейчас написано админ будет написано ученик или учитель. А еще так как сейчас я просматриваю свой профиль, то у меня есть кнопки редактирования теста и профиля. Если выйти из аккаунта, или зайти с другого - они пропадут. Из особенностей - можно удаленно выходить с других устройств, например с телефона можно выйти со школьного компьютера.

7. Конструктор тестов. Самая главная и самая сложная часть проекта. Вверху мы можем переключать задания и добавлять новые, у кждого задания нет определенного типа, их можно комбинировать из разных вопросов. Баллы начисляются за каждый вопрос а не задание целиком.

8. На странице теста отображается последний результат и лучший результат и все остальные решения. Если вы уже начали тест, то вы не можете начать другой, высветиться уведомление. Справа представлены все имеющиеся типы заданий. При решении мы не можем переключать вопросы, решать можно только по порядку или пропускать задания.

9. Результат теста отображается как у ученика так и учителя. Ученик на странице решения после прохождения может увидеть правильные ответы, снизу есть перкелючатель правильных ответов и ответов ученика.
